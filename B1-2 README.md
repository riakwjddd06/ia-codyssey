# 브랜드 아이덴티티
•	브랜드명: 픽사 애니메이션 스튜디오 (PIXAR)

•	타겟 (Target):
⁃	메인 타겟: 상상력이 풍부한 어린이 및 10대 청소년 (알파세대 및 Z세대)
⁃	서브 타겟: 바쁜 일상 속 동심을 잊고 살아가는 어른들 (2040세대)을 포함한 전 연령층

•	톤앤매너 (Tone & Manner):
⁃	따뜻한(Warm), 마법 같은(Magical), 경이로운(Cinematic), 향수를 불러일으키는(Nostalgic)
⁃	시각적 표현: 초반 어두운 배경과 대비되는 따뜻한 조명이 켜지며 시작(룩소 라이트), 픽사 대표작 속 캐릭터를 표현한 3D 오브제

•	USP (Unique Selling Proposition - 브랜드만의 강점):
⁃	상상력을 바탕으로, 평범한 일상 사물 혹은 상상에만 존재하는 캐릭터에 생명력과 감정을 불어넣는 압도적이고 독보적인 3D 비주얼과 모션
⁃	철학적 메세지와 감동적 스토리텔링으로 아이뿐 아닌 어른까지 공감할 수 있는 콘텐츠

# 캠페인 목표
•	단 10초의 짧은 시간 안에 픽사 특유의 동심으로 돌아가는 따뜻한 감성과 상상이 현실이 되는 기술 구현을 시각과 청각으로 몰입감 있게 전달하여, 관객들에게 따뜻한 향수와 기대감을 불러일으키는 브랜드 필름 제작.

# 핵심 메시지 (Core Message):
•	"잠들었던 상상이 현실로 깨어나는 곳, 픽사." (아이들에게는 상상이 눈앞에 펼쳐지는 놀이터, 어른들에게는 잊고 있던 순수함을 되찾는 공간이라는 이중적 의미 전달)
